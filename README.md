# Intruções para execução

API criada para consumir outra API e facilitar o consumo dos dados.

## Rodar
```shell
$ npm install
$ npm run start
```
## Faça uma requisição na url:
- http://localhost:3000/dsk?data_inicio=data-inicio&&data_fim=data-fim
